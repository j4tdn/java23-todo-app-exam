2. Lệnh gộp nhánh bug vào nhánh feature
git checkout feature
git merge bug
3. Lệnh gộp nhanh bug và nhánh feature dùng rebase
git checkout feature
git rebase bug
4. Phân biệt git merge và git rebase

8. Git log dùng để:
Git log dùng để xe lịch sử các commit
Xem HEAD đang trỏ đến commit nào

9.Git status dùng để:
Git status dùng để xem trạng thái của các file

10. Git add dùng để:
Git add dùng để thêm file và đưa file từ untracked sang staged

11. Git commit dùng để:
Git commit dùng để viết message và đưa các file từ trạng thái staged sang unmodified 

12. Mô tả lệnh để tạo branch ở local:
Tạo từ HEAD > git branch feature
Tạo từ nhánh master > git branch feature master

13. Mô tả lệnh để xóa branch ở local:
Git branch -d <tên nhánh>

14. Mô tả lệnh để xóa branch ở remote
Git push origin -d <tên nhánh ở remote>

15. Nhánh trung gian là gì
Nhánh trung gian là nhánh đại diện cho nhánh ở remote tại local 
nhánh trung gian kh thể sửa - thêm - bớt Commits
